== Description ==

Share-one-Drive is the #1 Ultimate OneDrive & SharePoint plugin for WordPress plugin on the market and part of a series of Cloud Plugins.

This plugin will help you to easily integrate OneDrive Personal, OneDrive Business and SharePoint Libraries into your WordPress website or blog. Share-one-Drive allows you to view, download, delete, rename files & folders stored in the cloud directly from a WordPress page. You can use Share-one-Drive as a File browser, Gallery, Audio- or Video-Player!
