=== WooCommerce Print Invoices/Packing Lists ===
Author: skyverge
Tags: woocommerce
Requires at least: 5.6
Tested up to: 6.1.1
Requires PHP: 7.4

This plugin adds order invoice, packing list, and pick list printing support to WooCommerce.

== Installation ==

1. Upload the plugin folder to the '/wp-content/plugins/' directory.

2. Activate the plugin through the 'Plugins' menu in WordPress.

== Usage ==

Go to WooCommerce > Settings > Invoices/Packing Lists, configure and start printing. Simple!

== Custom templates ==

1. All of the extension's templates are found in the 'templates/pip' directory. Copy any templates you wish to edit and place them in the '/wp-content/themes/yourtheme/woocommerce/pip/' directory.

2. Edit the templates as you wish.


== Upgrade Notice ==

= 3.0.0 =
3.0.0 is a major update so it is important that you make backups and ensure that you have read the documentation before updating. You may want to double check the extension's settings after upgrade. If you were using custom templates, you will need to update them with the newly introduced ones.
